# Unified Farm BLM Website

Official website for Unified Farm BLM - a family-run poultry farm in Mpigi, Uganda.

## Live Site

https://www.unifiedfarmblm.com

## Pages

- **Home** - Product showcase and farm overview
- **About Us** - Our story and mission
- **Blog** - Poultry farming guides and tips
- **Contact** - Get in touch with us
- **Privacy Policy** - How we handle your data
- **Terms of Service** - Website terms and conditions

## Built With

- HTML5
- CSS3 (no frameworks)
- Static site (GitHub Pages)
